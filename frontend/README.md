# PDF Cover Frontend

A React frontend for the PDF Cover Service. Upload PDFs and extract accessible cover pages.

## Features

- Drag & drop file upload
- Multiple file processing
- Batch download
- Configurable API endpoint
- Dark theme UI with MUI

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Render

1. Push to GitHub
2. Create a new **Static Site** on Render
3. Connect the repository
4. Set **Build Command**: `npm install && npm run build`
5. Set **Publish Directory**: `dist`
6. Deploy
